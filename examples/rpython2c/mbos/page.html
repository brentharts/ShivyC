mbos

This page was parsed into a DOM and rendered onto bare-metal VGA text by a freestanding kernel booted with qemu -kernel. No libc, no OS.

What works

Basic block layout with word wrap at the 80 column margin, headings, paragraphs, links, and lists. The same Node tree shape as minibrowser's dom.py, so the renderer can later be generated from rpython via py2c.
